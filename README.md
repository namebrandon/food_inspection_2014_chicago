food_inspection_2014_chicago
============================

City of Chicago 2014 Food Inspection Database as referenced in the blog post [The Dirtiest Restaurants in Chicago (2014 Edition)](http://brandonharris.io/dirtiest-restaurants-in-chicago/).

This database script was generated by MySQL Workbench 6.2 and should work on any recent version of MySQL Server. I have not tested it, merely exported it as there were requests for the data.
